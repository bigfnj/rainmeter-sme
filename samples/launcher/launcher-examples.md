## Curvs Circular Launcher (Lua-Driven Ring Layout)

```
REPO: udf/Curvs
TYPE: launcher (circular ring, Lua geometry, Shape buttons)
NOTES: Lua script calculates button positions around a circle.
       RingCount, Ring1.Count, Ring1.Size variables control geometry.
       Factory Lua generates Shape circles for each button from a template.
       MouseOver/click events delegated to the Lua controller.
       StyleButton MeterStyle applied to all button meters.
```

```ini
[Rainmeter]
Update=16
DynamicWindowSize=1

[Metadata]
Name=Curvs
Author=EyeZiS, juan_potato
License=Attribution-NonCommercial-ShareAlike 4.0 International

[Variables]
RingCount=1
RingStart=50
Ring1.Count=8
Ring1.Size=40
BorderColor=0,0,0,255
BorderSize=2
ButtonShift=5
@IncludeVariables=#@#Variables.inc
Offset=(#ButtonShift#+#BorderSize#/2)

[StyleButton]
OffsetTransform=Offset #Offset#,#Offset#
MouseOverAction=[!CommandMeasure "Controller" "onHover('#CURRENTSECTION#')"]
MouseLeaveAction=[!CommandMeasure "Controller" "onLeave('#CURRENTSECTION#')"]
LeftMouseUpAction=[!CommandMeasure "Controller" "onClick('#CURRENTSECTION#')"]

[StyleBorder]
OffsetTransform=Offset #Offset#,#Offset#
StyleAttributes=StrokeWidth #BorderSize# | Stroke Color #BorderColor# | Fill Color 0,0,0,1

[Rebuilder]
Measure=Script
ScriptFile=#@#Rebuilder.lua

[Controller]
Measure=Script
ScriptFile=#@#Curvs.lua

[Bounds]
Meter=Image
W=#CURRENTCONFIGWIDTH#
H=#CURRENTCONFIGHEIGHT#
DynamicVariables=1
SolidColor=0,0,0,1

; Generated by Rebuilder.lua from the ring layout:
; [Button0], [Button1], ... each positioned around the circle
; Each button is a Shape ellipse at the calculated X,Y position
; with the app icon as ImageName inside

; Example of what the generated .inc contains:
; [Button0]
; Meter=Shape
; MeterStyle=StyleButton | StyleBorder
; X=(CenterX + cos(0)*Radius - Size/2)
; Y=(CenterY + sin(0)*Radius - Size/2)
; Shape=Ellipse Size/2,Size/2,Size/2 | Fill Color 40,40,40,200
; Shape2=Ellipse Size/2,Size/2,Size/2 | StrokeWidth 2 | Stroke Color 60,60,60,200
; [Button0Icon]
; Meter=Image
; ImageName=#@#Icons\App0.png
; X=(CenterX + cos(0)*Radius - IconSize/2)
; Y=(CenterY + sin(0)*Radius - IconSize/2)
; W=IconSize
; H=IconSize
```

## Simple App Launcher Grid

```
PURPOSE: Template for a fixed-grid icon launcher.
         Left-click opens app. Right-click shows context menu with settings.
NOTES: Uses Image meters for icons and String meters for labels.
       ToolTipText shows app name on hover.
```

```ini
[Rainmeter]
Update=1000
DynamicWindowSize=1

[Variables]
IconSize=48
Cols=4
Rows=2
Padding=10
FontFace=Segoe UI
FontColor=255,255,255,200
FontSize=8
BGColor=20,20,20,200

[MeterBackground]
Meter=Shape
X=0
Y=0
Shape=RoundRectangle 0,0,((#IconSize#+#Padding#*2)*#Cols#+#Padding#),((#IconSize#+20+#Padding#*2)*#Rows#+#Padding#),8 | Fill Color #BGColor# | StrokeWidth 0

; Row 1
[MeterIcon1]
Meter=Image
ImageName=#@#Icons\Chrome.png
X=#Padding#
Y=#Padding#
W=#IconSize#
H=#IconSize#
PreserveAspectRatio=1
LeftMouseUpAction=["chrome.exe"]
ToolTipText=Google Chrome

[MeterLabel1]
Meter=String
X=((#IconSize#/2)+#Padding#)
Y=2R
StringAlign=CenterTop
FontFace=#FontFace#
FontSize=#FontSize#
FontColor=#FontColor#
AntiAlias=1
Text=Chrome

[MeterIcon2]
Meter=Image
ImageName=#@#Icons\Code.png
X=(#IconSize#+#Padding#*2)R
Y=(-#IconSize#)r
W=#IconSize#
H=#IconSize#
PreserveAspectRatio=1
LeftMouseUpAction=["code.exe"]
ToolTipText=Visual Studio Code

[MeterLabel2]
Meter=String
X=((#IconSize#/2)+(#IconSize#+#Padding#*2))r
Y=2R
StringAlign=CenterTop
FontFace=#FontFace#
FontSize=#FontSize#
FontColor=#FontColor#
AntiAlias=1
Text=VS Code

[MeterIcon3]
Meter=Image
ImageName=#@#Icons\Terminal.png
X=(#IconSize#+#Padding#*2)R
Y=(-#IconSize#)r
W=#IconSize#
H=#IconSize#
PreserveAspectRatio=1
LeftMouseUpAction=["wt.exe"]
ToolTipText=Windows Terminal

[MeterLabel3]
Meter=String
X=((#IconSize#/2)+(#IconSize#+#Padding#*2)*2)r
Y=2R
StringAlign=CenterTop
FontFace=#FontFace#
FontSize=#FontSize#
FontColor=#FontColor#
AntiAlias=1
Text=Terminal

[MeterIcon4]
Meter=Image
ImageName=#@#Icons\Explorer.png
X=(#IconSize#+#Padding#*2)R
Y=(-#IconSize#)r
W=#IconSize#
H=#IconSize#
PreserveAspectRatio=1
LeftMouseUpAction=["explorer.exe"]
ToolTipText=File Explorer

[MeterLabel4]
Meter=String
X=((#IconSize#/2)+(#IconSize#+#Padding#*2)*3)r
Y=2R
StringAlign=CenterTop
FontFace=#FontFace#
FontSize=#FontSize#
FontColor=#FontColor#
AntiAlias=1
Text=Explorer
```
